# PetFriends-API
Найти методы, которые ещё не реализованы в библиотеке, и написать их реализацию в файле api.py + 10 тест-кейсов.

Проект содержит библиотеку для работы с PetStore API и набор тестов.

## Структура
- `api/` - библиотека методов API
- `tests/` - позитивные и негативные тесты

## Запуск тестов
```bash
pytest tests/ -v
